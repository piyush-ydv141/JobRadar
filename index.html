About the Project

A practical guide for freshers, built by a fresher.

Why this project?

Most freshers apply to jobs without really knowing which skills are in demand, how salaries vary across cities, or which roles are realistic for 0–2 years of experience. This project turns raw job posting data into clear, visual answers.

As an MCA fresher, I wanted a portfolio project that feels close to real hiring problems. Recruiters can see both my technical skills (Python, SQL, dashboards) and my ability to think like an analyst.

What problem does it solve?

  • Helps freshers see which roles and skills are actually in demand.
  • Explains how salaries change by city, role, and tech stack.
  • Shows where to focus learning time (Python, SQL, data roles, etc.).
  • Gives recruiters a single project that demonstrates analytics thinking end-to-end.

Data Sources, Scope & Limitations

The analysis is inspired by public job posting datasets from platforms like Kaggle (e.g., Naukri / LinkedIn style exports) combined with realistic assumptions for 2024–2025. The focus is on fresher roles (0–2 years) in major Indian cities.

This is not a live feed of all jobs in India. Instead, it is a curated, cleaned snapshot that is sufficient to show trends, build dashboards, and demonstrate analytical thinking.

Data & Methodology

From raw job postings to clean, analyzable data.

Dataset Source

The dataset is based on public job posting data for India (Kaggle-style exports) with fields like job title, company, city, experience required, skills, and salary range.

Only roles clearly tagged as Entry Level / Fresher / 0–2 years were included in the final dataset.

Data Cleaning Steps

  • Duplicate removal – Dropped repeated postings by title + company + city.
  • Salary normalisation – Converted CTC ranges into annual LPA mid-points.
  • Skill extraction – Tokenised skills (Python, SQL, Java, Excel, etc.) into a skill matrix.
  • Fresher filtering – Kept only jobs with 0–2 years required experience.

Tools & Workflow

  • Python (Pandas, NumPy) – Exploratory data analysis, cleaning, feature engineering.
  • SQL – Aggregations such as salary by role, skill frequency, city-wise hiring.
  • Power BI Concept / Chart.js – Interactive visualisations and dashboard layout.

The same logic used for Power BI measures is replicated here in JavaScript to make the dashboard fully web-based and portfolio friendly.

Insights & Findings

Plain-language explanations backed by the dashboard data.

Python + SQL premium

Fresher roles that require both Python and SQL offer around 25–35% higher median salary compared to generic IT support or non-technical roles.

Evidence: Higher LPA ranges for Data Analyst / Data Engineer / BI roles concentrated in Bangalore, Hyderabad, and Gurgaon.

Bangalore & Hyderabad lead hiring

The highest concentration of fresher openings appear in Bangalore and Hyderabad, followed by Pune, Chennai, and NCR.

Evidence: City-wise posting counts show a clear skew towards Tier-1 tech hubs.

Data roles: skills over years

Many data roles (Data Analyst, BI Analyst, Junior Data Engineer) accept freshers with 0–1 year experience, but the skill expectations are higher (Python, SQL, Excel, visualization tools).

Evidence: Data roles cluster at 0–2 years but require more technical skills per posting.

Full stack vs pure backend

Entry-level full stack developer roles often pay slightly higher than pure front-end roles, especially when paired with modern frameworks and cloud exposure.

Non-metro tradeoff

Jobs in non-metro cities exist but typically come with lower salary bands and fewer data / product roles. They can still be a good option to gain initial experience quickly.

Career Recommendations for Freshers

Actionable next steps based on the analysis.

Skills to Focus On

  • Core Programming – Python or Java for backend / data roles.
  • Data Skills – SQL, Excel, and at least one BI tool (Power BI, Tableau).
  • Web Fundamentals – HTML, CSS, JavaScript for developer roles.
  • Version Control – Git & GitHub for collaboration.
  • Communication – Writing clear summaries of findings (like this site).

Best Cities to Target

  • Primary: Bangalore, Hyderabad, Pune.
  • Secondary: Chennai, Gurgaon, Noida, Mumbai.
  • Strategy: Start with remote / hybrid roles in these hubs, then expand to product companies and startups.

Roles with Strong Growth

  • Data Analyst / Business Analyst
  • Junior Data Engineer
  • Full Stack Developer
  • Cloud / DevOps Associate (with certification)

These roles combine high demand, clear skill paths, and good long-term salary growth.

Salary Optimisation Tips

  • Combine Python + SQL + Excel/Power BI for data roles with better CTC.
  • Target Tier-1 cities first, then negotiate based on competing offers.
  • Showcase projects with real data (like this one) in your resume and LinkedIn.
  • Prepare for practical problem-solving interviews, not just theory.
